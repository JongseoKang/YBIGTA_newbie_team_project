# YBIGTA_newbie_team_project

## 실행 방법
```
git clone https://github.com/JongseoKang/YBIGTA_newbie_team_project
pip install -r requirements.txt
cd ./app
uvicorn main:app --reload
```
브라우저를 이용하여 http://127.0.0.1:8000/static/index.html 접속
